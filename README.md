## Installation

Appwritelite is simpler version of [Appwrite.io](https://appwrite.io). Running your server is as easy as running one command from your terminal. You can either run Appwrite on your localhost using docker-compose or on any other container orchestration tool like Kubernetes, Docker Swarm, or Rancher.
